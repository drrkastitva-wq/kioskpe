# Let’s Legal — MVP Roadmap

## Phase 0: Foundation (Week 1)
- Finalize requirements and roles
- Auth model + security baseline
- Data model V1
- UI wireframes for dashboard, case list, case details

## Phase 1: Case Ops Core (Weeks 2-4)
- New case intake form
- Case listing, filters, status, next hearing date
- Hearing reminders and follow-up tasks
- Advocate diary with calendar view
- Basic documents/templates module

## Phase 2: Legal Directories (Weeks 5-6)
- All India courts directory with location/contact
- Police station directory with location/contact
- Court calendar + holidays
- Bar association information

## Phase 3: Lawyer Verification + Public Access (Weeks 7-8)
- Public registration flow
- Bar Council ID + metadata capture
- Verification queue for admin
- Controlled access by verification status

## Phase 4: Practice Productivity (Weeks 9-10)
- Fee tracker
- Deadline/limitation calculator
- Cause list tracker
- Checklist-based matter workflow

## MVP Success Metrics
- 80% of active cases tracked in platform
- 90% hearing reminders delivered on time
- <2 min average time to create new case
- >70% weekly active advocates among registered lawyers
