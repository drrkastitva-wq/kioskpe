# Let’s Legal Backend

Node.js + Express + TypeScript API for the Let’s Legal app.

## Run

```powershell
npm install
npm run dev
```

Default server URL: `http://localhost:4000`

## Endpoints
- `GET /health`
- `GET /api/cases`
- `POST /api/cases`
- `PATCH /api/cases/:id`
- `DELETE /api/cases/:id`
- `GET /api/reminders`
- `POST /api/reminders`
- `PATCH /api/reminders/:id`
- `GET /api/diary`
- `POST /api/diary`
